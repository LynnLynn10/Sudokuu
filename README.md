# Sudoku Puzzle Generator
Made by Grace Kuang

A program that is meant to create a solved 9x9 Sudoku Puzzle in a reasonable amount of time.

To run the program, all that is needed is to paste the code into a compiler-since it's all in one file-and run it.

The board is generated by first filling the board will all 0s (to represent which cells are still empty); it then attempts to fill the very first cell with the first number in a randomized list of numbers from 1-9. It then moves on to the next cell, rerandomizing the list and again picking the first number. If the program determines that the number is not fit to be placed in a specific cell (if it breaks the rules of sudoku), it tries again with a different number be randomizing the list again and picking a different first number. Repeat until the board is filled.
